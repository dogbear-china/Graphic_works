基于 Taichi 实现的实时 Phong 光照模型演示程序，支持球体与圆锥的渲染，并可通过 GUI 动态调整材质参数。

##  项目介绍

本项目使用 **Taichi** 语言实现了一个简单的光线追踪渲染器，主要演示 **Phong 反射模型** 在实时图形学中的应用。程序通过 GPU 并行计算每个像素的颜色，支持球体和圆锥两种几何体的求交计算，并提供了交互式 GUI 面板，允许用户动态调整材质的四个关键参数，实时观察渲染效果的变化。

### 主要特性

- **GPU 加速渲染**：基于 Taichi 的 `@ti.kernel` 实现像素级并行计算
-  **多种几何体**：支持球体（Sphere）和圆锥（Cone）的精确求交
-  **Phong 光照模型**：完整实现环境光（Ambient）、漫反射（Diffuse）和高光（Specular）分量
-  **实时交互界面**：使用 Taichi 内置 GUI 系统，可动态调节材质参数
-  **直观的视觉反馈**：参数调整后渲染效果立即更新

### 核心概念

#### Phong 光照模型

Phong 模型将物体表面的光照分解为三个部分：
最终颜色 = 环境光 + 漫反射 + 高光
= Ka * Ia + Kd * (N·L) * Id + Ks * (R·V)^n * Is


其中：
- **Ka, Kd, Ks**：分别为环境光、漫反射、高光系数
- **n (Shininess)**：高光指数，控制高光区域的集中程度
- **N**：表面法线
- **L**：光线方向
- **V**：视线方向
- **R**：反射光线方向

#### 几何体求交

- **球体**：通过解一元二次方程 `|P - C|^2 = r^2` 计算交点
- **圆锥**：假设圆锥轴线与 Y 轴平行，通过圆锥方程 `x² + z² = k·y²` 推导求交公式
