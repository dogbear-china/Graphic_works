# 基于 Taichi 的高性能贝塞尔曲线实时绘制系统

## 文件名称
`bezier_curve_gpu.py`

## 代码概述
这段代码实现了一个**基于 GPU 加速的贝塞尔曲线实时绘制系统**，主要功能包括：
- 通过鼠标点击交互式添加控制点
- 使用 De Casteljau 算法计算贝塞尔曲线路径
- 利用 Taichi GPU 后端实现高性能像素渲染
- 实时显示控制点、控制多边形和曲线

## 适用场景
- 交互式曲线设计与教学演示
- 计算机图形学课程实验
- 需要实时反馈的贝塞尔曲线调试工具
- GPU 编程与性能优化案例学习
